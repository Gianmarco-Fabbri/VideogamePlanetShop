# Videogame Planet Shop

Prima di eseguire l'applicativo è necessario popolare il database GameUniverse.
Si devono aprire i file collocati nella directory "DataBase" all'interno di un collegamento
creato nell'applicazione MySQL Workbench.
Una volta creato e popolato il database si deve eseguire il file main.py che avvia l'applicazione,
ci sono due maniere:
-Aprire un terminale e collocarsi all'interno della directory "CodiceApplicativo" e digitare
il comando "python main.py".
-Aprire la cartella "CodiceApplicativo" in un IDE come Visual Studio Code ed eseguire il file di avvio.

Una volta avviata si presenterà la finestra di login in cui sarà possibile inserire email e password di un utente.
